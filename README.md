# Pixel Zone

**موقع تفاعلي للأولاد والبنات يحتوي على:**

- دردشة مباشرة مع أسماء المستخدمين الموحدة (UserXXXX).  
- الرد على مستخدم معين عبر الضغط على اسمه.  
- قسم للرسم المحمي (يمكن الرسم داخل الكانفس فقط).  
- لعبة XO ضد الذكاء الاصطناعي، تتفضى المربعات عند الانتهاء.  
- التصميم متجاوب ويعمل على كل الأجهزة.

---

## Features (English)

- Real-time chat with fixed random usernames (UserXXXX).  
- Ability to reply to a specific user by clicking on their name.  
- Protected drawing canvas.  
- XO game against AI; clears grid after someone wins.  
- Responsive design, works on all devices.

---

## Usage

1. افتح الموقع، اضغط على الباب الكبير للدخول.  
2. اختر قسم الأولاد أو البنات.  
3. يمكنك كتابة اسمك أو استخدام الاسم العشوائي الموحد.  
4. ابدأ الدردشة أو الرسم أو لعب XO.  

---

## Notes

- المشروع يستخدم Firebase Realtime Database لتخزين الرسائل مباشرة.
